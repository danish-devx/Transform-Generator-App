## 📌 Project Overview
**Transform Generator App** is an interactive frontend tool built using **HTML, CSS, and JavaScript**.  
It allows users to apply and preview CSS `transform` properties like rotate, scale, skew, and translate in real time.

This project showcases DOM manipulation, dynamic input handling, and updating CSS styles using JavaScript.
